## Array of products

#### Problem Statement


Given an integer array `nums`, return an array `answer` such that `answer[i]` is equl to the product of all the elements of `nums` exept `nums[i]`.


#### Complexity Analysis

- Time Complexity: **O(N)**
- Space Complexity: **O(N)**


#### For example:

```python
Input: [1, 2, 3, 4]
Output: [24, 12, 8, 6]
```

or

```python
Input: [-1, 1, 0, -3, 3]
Output: [0, 0, 9, 0, 0]
```
