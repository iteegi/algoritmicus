## Move element to end

#### Problem Statement


Move the given number to the end of the array.


#### Complexity Analysis

- Time Complexity: **O(N)**
- Space Complexity: **O(1)**


#### For example:

```
array: [1, 3, 2, 4, 2, 2, 2, 7]
number: 2
Output: [1, 3, 7, 4, 2, 2, 2, 2]
```
